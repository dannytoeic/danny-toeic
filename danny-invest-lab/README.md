# 데니 투자연구소

10억 프로젝트 자문 시스템 MVP입니다.

이 앱은 자동매매 시스템이 아니며, 매수·매도 주문 실행 기능을 포함하지 않습니다. 표시되는 내용은 투자 판단 보조용이고 최종 결정은 사용자가 직접 수행합니다.

## 실행

```bash
npm install
npm run dev
```

## 구현 범위

- 계좌별 현금과 보유종목 수동 입력
- 총자산, 현금, 투자금, 방어현금 목표, 추가 투입 가능금 자동 계산
- 반도체, 미국 ETF, CMA, 연금 비중 계산
- 1개월/3개월/6개월 고점, 20일선, 60일선 수동 입력 기반 눌림 판단
- 매수 계획 분할 방식과 S&P500 60%, 나스닥100 40% 기본 배분
- 알림 조건 화면과 현재 알림 산출
- 오늘의 판단 문장과 ChatGPT 자문 요청문 생성
- Supabase `invest_` prefix schema, RLS, authenticated GRANT SQL
